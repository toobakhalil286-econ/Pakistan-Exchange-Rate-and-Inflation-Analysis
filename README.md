# 📊 Exchange Rate Depreciation and Inflation in Pakistan (2020–2025)

## 📖 Project Overview

This independent research project examines the relationship between Pakistan's USD/PKR exchange rate and inflation from January 2020 to June 2025.

The project uses Microsoft Excel to clean, analyze, and visualize macroeconomic data through descriptive statistics, correlation analysis, annual summaries, and an interactive dashboard.

---

## 🎯 Research Objectives

- Analyze monthly exchange rate trends
- Analyze monthly inflation trends
- Calculate descriptive statistics
- Measure the correlation between exchange rate and inflation
- Develop an interactive Excel dashboard

## 📈 Key Findings

- Exchange rate depreciation accelerated during 2022–2023.
- Inflation followed a similar upward trend before easing in 2024–2025.
- Pearson Correlation Coefficient: **0.423**
- The analysis indicates a moderate positive relationship between exchange rate depreciation and inflation.
